# Cara Login ke SEKAR

Panduan lengkap untuk login ke aplikasi mobile SEKAR dan dashboard web.

## Login di Aplikasi Mobile

<p align="center"><img src="/img/mobile/login.png" alt="Layar login aplikasi" width="280" /></p>

*Layar login aplikasi SEKAR.*

### Langkah-langkah Login

1. **Buka Aplikasi SEKAR**
   - Ketuk ikon aplikasi SEKAR di layar utama perangkat Anda

2. **Isi Nomor Telepon**
   - Masukkan nomor telepon Anda (format: 081234567890)
   - Pastikan nomor sesuai dengan yang terdaftar di sistem

3. **Isi Kata Sandi**
   - Masukkan kata sandi Anda
   - Ketuk ikon mata untuk menampilkan/menyembunyikan kata sandi

4. **Login**
   - Ketuk tombol **MASUK** untuk login
   - Tunggu aplikasi memproses login (biasanya 2-3 detik)

5. **Berhasil**
   - Anda akan diarahkan ke halaman beranda aplikasi
   - Status koneksi ditampilkan di atas layar

:::tip
**Ingat Saya?** Centang kotak "Ingat Saya" untuk login otomatis berikutnya.
:::

## Login di Dashboard Web

![Dashboard web SEKAR](/img/web/login.png)

*Halaman login dashboard web.*

### Langkah-langkah Login Web

1. **Buka Browser**
   - Buka browser di komputer atau tablet Anda

2. **Kunjungi Dashboard**
   - Masukkan alamat: **sekar.wahyutrip.com**

3. **Pilih Metode Login**
   - Login dengan **nama pengguna** atau **nomor telepon**

4. **Isi Kredensial**
   - **Nama Pengguna/Nomor Telepon**: Masukkan username atau nomor telepon
   - **Kata Sandi**: Masukkan kata sandi Anda

5. **Klik Login**
   - Klik tombol **MASUK**

6. **Berhasil**
   - Anda akan melihat dashboard sesuai peran Anda

:::warning
**HTTPS Wajib**: Dashboard web hanya bisa diakses melalui koneksi internet yang aman (HTTPS). Pastikan koneksi stabil.
:::

## Lupa Kata Sandi?

### Cara Reset Kata Sandi

Jika Anda lupa kata sandi, ikuti langkah berikut:

1. **Di Aplikasi Mobile:**
   - Ketuk **"Lupa Kata Sandi?"** di layar login
   - Masukkan nomor telepon Anda
   - Ikuti instruksi yang dikirim via SMS atau hubungi admin

2. **Di Dashboard Web:**
   - Klik **"Lupa Kata Sandi?"** di halaman login
   - Masukkan nama pengguna atau nomor telepon
   - Tunggu instruksi dari admin atau cek email

:::note
Saat ini, proses reset kata sandi harus dilakukan melalui admin. Hubungi kepala rayon atau admin data Anda untuk direset.
:::

## Ganti Kata Sandi

### Mengubah Kata Sandi Anda

**Dari Aplikasi Mobile:**

1. Buka tab **Profil** (di bawah)
2. Ketuk **Pengaturan** (ikon roda gigi)
3. Pilih **Ganti Kata Sandi**
4. Masukkan:
   - Kata sandi lama
   - Kata sandi baru
   - Konfirmasi kata sandi baru
5. Ketuk **Simpan**

**Dari Dashboard Web:**

1. Klik avatar Anda di sudut atas kanan
2. Pilih **Pengaturan**
3. Buka tab **Keamanan**
4. Klik **Ubah Kata Sandi**
5. Masukkan kata sandi lama dan baru
6. Klik **Simpan**

:::warning
**Pilih kata sandi yang kuat** dengan kombinasi huruf, angka, dan simbol. Jangan bagikan kata sandi dengan siapa pun.
:::

## Logout

### Cara Keluar

**Dari Aplikasi Mobile:**

1. Buka tab **Profil**
2. Scroll ke bawah
3. Ketuk **KELUAR**
4. Konfirmasi keluar

**Dari Dashboard Web:**

1. Klik avatar di sudut atas kanan
2. Pilih **Keluar**
3. Anda akan dialihkan ke halaman login

## Troubleshooting

### Masalah yang Sering Terjadi

**"Login gagal - Periksa nomor telepon dan kata sandi"**
- Pastikan nomor telepon dan kata sandi benar
- Coba login lagi setelah beberapa detik
- Hubungi admin jika akun belum didaftarkan

**"Tidak bisa login di WiFi publik"**
- WiFi publik mungkin memblokir akses
- Coba gunakan data seluler (4G/5G)
- Hubungi admin WiFi untuk membuka akses

**"Aplikasi mengatakan 'Offline'"**
- Pastikan perangkat terhubung ke internet
- Nyalakan kembali WiFi atau data seluler
- Restart aplikasi SEKAR

**"Login berhasil tapi tidak muncul halaman berikutnya"**
- Tunggu 5-10 detik untuk loading
- Periksa koneksi internet
- Restart aplikasi
- Hubungi support jika berlanjut
