# Cara Instalasi Aplikasi Mobile SEKAR

Panduan langkah demi langkah untuk mengunduh dan memasang aplikasi SEKAR di perangkat Android Anda.

## Persyaratan Perangkat

### Spesifikasi Minimum

- **OS**: Android 8.0 atau lebih baru
- **RAM**: 2 GB minimal (4 GB disarankan)
- **Penyimpanan**: 100 MB ruang kosong
- **Koneksi**: WiFi atau data seluler 4G/5G

### Izin yang Diperlukan

Aplikasi SEKAR memerlukan beberapa izin untuk berfungsi dengan baik:

- **Lokasi (GPS)** — Untuk tracking real-time dan verifikasi GPS
- **Kamera** — Untuk foto verifikasi saat absen dan laporan kerja
- **Notifikasi** — Untuk menerima update dan reminder
- **Penyimpanan** — Untuk menyimpan data dan foto

## Langkah-Langkah Instalasi

### 1. Unduh File APK

:::note
Unduh dari halaman resmi SEKAR untuk memastikan Anda mendapat versi terbaru.
:::

**Cara Mengunduh:**

1. Buka browser di perangkat Android Anda (Chrome, Firefox, atau browser lainnya)
2. Kunjungi: **https://sekar.wahyutrip.com/android**
3. Klik tombol **"Unduh APK"** atau **"Download"**
4. File akan mulai diunduh (biasanya 50-100 MB)
5. Tunggu hingga unduhan selesai

:::tip
Jika unduh lambat, pastikan koneksi internet stabil atau gunakan WiFi.
:::

### 2. Persiapan Instalasi

**Izinkan Aplikasi dari Sumber Tidak Dikenal:**

Karena file APK tidak berasal dari Google Play Store, Anda perlu memberikan izin:

1. Buka **Pengaturan** > **Keamanan** atau **Privasi**
2. Cari opsi **"Aplikasi dari Sumber Tidak Dikenal"** atau **"Install Unknown Apps"**
3. Pilih aplikasi **File Manager** atau **Chrome** (aplikasi yang Anda gunakan untuk download)
4. Aktifkan izin **"Izinkan dari sumber ini"** atau **"Allow from this source"**

### 3. Instal Aplikasi

1. Buka **File Manager** atau **Downloads** di perangkat Anda
2. Cari file yang bernama `sekar-*.apk` (misalnya: `sekar-1.0.0.apk`)
3. Ketuk file tersebut untuk memulai instalasi
4. Klik **"Instal"** atau **"Install"**
5. Tunggu hingga instalasi selesai (biasanya 30-60 detik)
6. Klik **"Buka"** atau **"Open"** untuk meluncurkan aplikasi

### 4. Konfigurasi Awal

**Login Pertama Kali:**

1. Aplikasi SEKAR akan membuka halaman login
2. Masukkan nomor telepon dan kata sandi Anda
3. Klik **"MASUK"**

**Berikan Izin:**

Saat pertama kali login, aplikasi akan meminta beberapa izin:

- **Izin Lokasi (GPS)**: Ketuk **"Izinkan"** untuk mengaktifkan GPS
- **Izin Kamera**: Ketuk **"Izinkan"** untuk menggunakan kamera
- **Izin Notifikasi**: Ketuk **"Izinkan"** untuk menerima notifikasi

:::warning
**Izin ini penting untuk fungsi aplikasi.** Jika Anda tidak memberikan izin sekarang, Anda tidak bisa menggunakan fitur tertentu seperti absen, laporan kerja, atau menerima notifikasi.
:::

### 5. Verifikasi Instalasi

Setelah login berhasil:

1. Anda akan melihat halaman **"Beranda"** atau **"Home"**
2. Cek status koneksi di atas layar (harus menunjukkan **"ONLINE"** dengan tanda hijau)
3. Aplikasi siap digunakan!

## Mengaktifkan GPS

### GPS Sangat Penting untuk SEKAR

Aplikasi SEKAR **memerlukan GPS aktif** untuk:

- Verifikasi lokasi saat absen
- Pelacakan real-time pekerja
- Laporan kerja dengan lokasi akurat

### Cara Mengaktifkan GPS

1. Buka **Pengaturan** > **Lokasi**
2. Aktifkan **"Lokasi"** atau **"Location Services"**
3. Pilih mode lokasi: **"Akurasi Tinggi"** atau **"High Accuracy"** (kombinasi GPS + WiFi)
4. Buka kembali aplikasi SEKAR

:::note
**Lebih Hemat Baterai?** Mode "Hemat Daya" menggunakan WiFi dan data seluler saja (GPS mati). Tapi ini membuat akurasi GPS berkurang. Gunakan **"Akurasi Tinggi"** untuk hasil terbaik.
:::

## Update Aplikasi

### Cara Update ke Versi Terbaru

1. Kunjungi halaman: **https://sekar.wahyutrip.com/android**
2. Cek versi aplikasi yang Anda gunakan di bagian **"Profil"** > **"Diagnostik"**
3. Jika ada versi lebih baru di halaman, unduh dan instal file APK baru
4. Proses instalasi akan menimpa versi lama secara otomatis

:::tip
Aplikasi akan mengingatkan Anda jika ada versi terbaru tersedia. Selalu update untuk fitur dan keamanan terbaru.
:::

## Uninstal Aplikasi

Jika Anda ingin menghapus aplikasi SEKAR:

1. Buka **Pengaturan** > **Aplikasi**
2. Cari **"SEKAR"** di daftar aplikasi
3. Ketuk aplikasi, lalu klik **"Uninstal"**
4. Konfirmasi penghapusan
5. Data aplikasi akan dihapus dari perangkat

:::warning
**Catatan Penting**: Uninstal aplikasi akan menghapus semua data lokal yang disimpan. Jika Anda punya laporan atau data offline yang belum disinkronkan, pastikan sudah terupload ke server sebelum uninstal.
:::

## Troubleshooting Instalasi

### "File APK Rusak atau Tidak Valid"

**Solusi:**
- Hapus file APK yang sudah diunduh
- Unduh kembali dari halaman resmi
- Pastikan unduhan selesai 100% sebelum instalasi
- Coba unduh menggunakan WiFi daripada data seluler

### "Tidak Punya Cukup Ruang Penyimpanan"

**Solusi:**
- Buka **Pengaturan** > **Penyimpanan**
- Hapus aplikasi atau file yang tidak perlu
- Kosongkan cache aplikasi lain
- Coba instalasi lagi

### "Izin Aplikasi dari Sumber Tidak Dikenal Diblokir"

**Solusi:**
- Buka **Pengaturan** > **Keamanan**
- Aktifkan **"Aplikasi dari Sumber Tidak Dikenal"**
- Pilih aplikasi browser yang Anda gunakan
- Coba instal kembali

### "Aplikasi Crash atau Tidak Bisa Dibuka"

**Solusi:**
- Restart perangkat Anda
- Hapus dan instal ulang aplikasi SEKAR
- Pastikan Android Anda up-to-date
- Hubungi admin jika masalah berlanjut
