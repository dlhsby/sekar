# Absen Masuk & Keluar dengan Foto Verifikasi

Panduan untuk melakukan absen (clock in/out) di aplikasi SEKAR dengan foto dan verifikasi GPS.

## Mengapa Absen Penting?

Absen mencatat:
- **Waktu Masuk**: Kapan Anda mulai bekerja
- **Waktu Keluar**: Kapan Anda selesai bekerja
- **Lokasi GPS**: Di mana Anda saat absen
- **Foto Verifikasi**: Bukti Anda benar-benar di lokasi

Semua data ini akan ditampilkan di laporan dan dashboard supervisor.

## Persyaratan untuk Absen

:::warning
Pastikan sebelum absen:

1. **GPS Aktif** — Perangkat harus menangkap sinyal GPS
2. **Kamera Berfungsi** — Untuk mengambil foto verifikasi
3. **Koneksi Internet** — WiFi atau data seluler aktif
4. **Lokasi Benar** — Anda harus berada dalam radius ±100m dari area yang ditugaskan
:::

## Langkah-Langkah Absen Masuk

### 1. Buka Layar Absen

**Metode 1 - Dari Beranda:**
- Buka tab **Beranda** (Home)
- Ketuk tombol besar **"MASUK"** atau **"Absen Masuk"**

**Metode 2 - Dari Menu:**
- Buka tab **Menu** (☰)
- Ketuk **"Kehadiran"** → **"Absen"** atau langsung tap **"MASUK"**

### 2. Periksa Informasi Shift

Layar akan menampilkan:

```
┌─────────────────────────────┐
│   INFORMASI KEHADIRAN      │
├─────────────────────────────┤
│ Waktu Sekarang             │
│   Senin, 24 Juni 2026      │
│   07:15                    │
│                            │
│ Jadwal Shift               │
│   Shift 1: 06:00 - 14:00   │
│                            │
│ Area Ditugaskan            │
│   Taman Bungkul            │
│                            │
│ Koordinat GPS              │
│   -7.295479, 112.762227    │
│   Radius: 100m             │
│                            │
│ [AMBIL SELFIE]             │
└─────────────────────────────┘
```

**Periksa:**
- ✅ Jadwal sudah dimulai (belum lewat dari jam masuk)
- ✅ Anda berada di area yang ditugaskan
- ✅ Koordinat GPS menunjukkan lokasi yang benar

:::note
**Status Keterlambatan**: Jika Anda masuk lebih lambat dari jam shift, akan ada badge **"Terlambat"** di warna merah.
:::

### 3. Ambil Foto Selfie

1. Ketuk tombol **"AMBIL SELFIE"**
2. Izinkan akses kamera jika diminta
3. Ambil foto diri Anda (wajah jelas dan terlihat)
4. Periksa foto — jika tidak puas, ambil lagi
5. Ketuk **"Gunakan Foto Ini"** untuk menyetujui

:::tip
**Tips Foto Terbaik:**
- Cahaya cukup (di siang hari, hindari silauan matahari langsung)
- Wajah jelas dan terlihat
- Background bisa area kerja atau lokasi apapun
- Pose santai, bukan pose aneh
:::

### 4. Konfirmasi & Submit

1. Periksa sekali lagi:
   - Foto sudah benar
   - Jam sudah tepat
   - Lokasi sudah benar
2. Ketuk **"KONFIRMASI MASUK"** atau **"MASUK"**
3. Tunggu sistem memproses (biasanya 2-3 detik)

### 5. Berhasil!

Anda akan melihat notifikasi:
```
✓ Absen Masuk Berhasil
  Waktu: 07:15
  Lokasi: Taman Bungkul
```

Kembali ke Beranda untuk melihat jam masuk tercatat.

## Langkah-Langkah Absen Keluar

### Proses Absen Keluar Serupa dengan Masuk

1. **Buka Layar Absen**
   - Tab Beranda → Ketuk **"KELUAR"**
   - Atau Menu → Kehadiran → **"KELUAR"**

2. **Periksa Informasi**
   - Pastikan Anda masih di area kerja
   - Jam keluar sudah cukup (minimal jam kerja tercapai)

3. **Ambil Foto Selfie Lagi**
   - Foto verifikasi sebelum keluar
   - Sama seperti proses masuk

4. **Konfirmasi Keluar**
   - Klik **"KONFIRMASI KELUAR"** atau **"KELUAR"**
   - Tunggu proses selesai

5. **Selesai!**
   - Anda telah absen keluar
   - Jam kerja tercatat di sistem

:::warning
**Perhatian Penting:**

- Jika Anda **keluar sebelum jam kerja normal**, itu akan tercatat sebagai "Keluar Awal" (berpengaruh pada penilaian)
- Pastikan supervisor sudah **menyetujui keluar awal** jika ada keperluan
:::

## Keterlambatan (Terlambat)

### Apa itu Terlambat?

Anda dianggap **Terlambat** jika absen masuk **lebih lambat** dari jam shift yang dijadwalkan.

**Contoh:**
- Jadwal Shift 1: 06:00 - 14:00
- Anda absen masuk jam: 06:45
- **Status: TERLAMBAT** (45 menit terlambat)

### Indikator Keterlambatan

Status akan ditampilkan dengan warna merah (**🔴 TERLAMBAT**) di:
- Kartu Kehadiran di Beranda
- Layar Absen Masuk
- Laporan Kehadiran

### Apakah Terlambat Berpengaruh?

Ya, keterlambatan akan:
- ✓ Tercatat di sistem
- ✓ Terlihat di laporan supervisor
- ✓ Mempengaruhi penilaian kinerja

### Bagaimana Menjelaskan Keterlambatan?

Hubungi supervisor/korlap Anda untuk menjelaskan alasan terlambat. Mereka bisa:
- Mencatat alasan di sistem
- Menyetujui atau menolak keterlambatan
- Memberikan rekomendasi

## Offline Mode (Tanpa Internet)

### Absen Saat Offline

Jika koneksi internet terputus:

1. Anda **tetap bisa absen** (foto akan disimpan lokal)
2. Sistem akan coba upload saat koneksi kembali
3. Status akan menunjukkan **🟡 SINKRONISASI**

:::warning
Pastikan koneksi kembali **dalam 24 jam** agar data terupload dengan benar.
:::

### Cara Menyinkronkan Data

Jika data offline belum terupload:

1. Pastikan koneksi internet aktif
2. Buka aplikasi SEKAR
3. Tunggu notifikasi "Sinkronisasi Selesai"
4. Atau buka **Menu** → **Profil** → **Diagnostik** untuk cek status

## Troubleshooting Absen

### "GPS Tidak Akurat / Diluar Radius"

**Masalah**: Pesan error "Lokasi Anda diluar area yang ditugaskan"

**Penyebab Kemungkinan:**
- GPS belum fix (sinyal terlalu lemah)
- Benar-benar berada di luar radius 100m

**Solusi:**
1. Tunggu 1-2 menit agar GPS lebih akurat
2. Pastikan Anda fisik di lokasi area kerja
3. Coba beberapa meter lebih dekat ke pusat area
4. Jika masih gagal, hubungi supervisor

:::note
Sistem **tidak akan memblokir absen**, tapi data GPS akan tercatat dan supervisor akan melihatnya di monitoring.
:::

### "Foto Tidak Bisa Diambil"

**Masalah**: Aplikasi tidak bisa akses kamera

**Solusi:**
1. Cek izin kamera di Pengaturan > Izin Aplikasi
2. Pastikan izin "Kamera" untuk SEKAR sudah aktif
3. Restart aplikasi
4. Jika masih gagal, hubungi IT/admin

### "Absen Gagal Upload"

**Masalah**: "Gagal mengirim data, cek koneksi internet"

**Solusi:**
1. Periksa koneksi WiFi atau data seluler
2. Tunggu beberapa saat dan coba lagi
3. Jika offline, data akan upload saat koneksi kembali
4. Hubungi admin jika terus gagal

### "Sudah Absen Masuk Tapi Tidak Terlihat di Beranda"

**Solusi:**
1. Tutup aplikasi sepenuhnya (swipe up atau gunakan recent apps)
2. Buka kembali aplikasi
3. Buka tab Beranda
4. Data akan refresh
5. Jika masih tidak muncul, hubungi admin

## Tips & Trik Absen

:::tip
**1. Absen Tepat Waktu** — Datang lebih awal 5-10 menit sebelum shift mulai untuk menghindari keterlambatan.

**2. Foto Jelas** — Ambil foto dengan wajah jelas, bukan foto burur atau silhouette.

**3. Pantau GPS** — Jika bekerja di area berbeda, pastikan Anda berada di lokasi yang ditugaskan saat absen.

**4. Sinkronisasi Rutin** — Jika sering offline, pastikan aplikasi tersinkronisasi setiap akhir hari.
:::
