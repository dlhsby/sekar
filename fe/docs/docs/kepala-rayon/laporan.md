# Laporan dan Analitik Kinerja Rayon

Panduan untuk Kepala Rayon melihat laporan dan analitik kinerja rayon Anda.

## Apa itu Dashboard Kepala Rayon?

Dashboard Kepala Rayon menampilkan:

- **Ringkasan Kinerja** — Overview kehadiran, produktivitas, kualitas
- **Laporan Detail** — Per pekerja, per area, per periode
- **Analitik Grafis** — Trend kinerja, perbandingan target vs realisasi
- **Export Data** — Untuk laporan resmi ke manajemen

---

## Membuka Dashboard

1. Login ke dashboard web dengan akun Kepala Rayon Anda
2. Anda akan melihat dashboard home dengan cards ringkasan

```
├─────────────────────────────────────┤
│                                     │
│ Kehadiran              │ Ketelitian │
│ ████████░░ 85%       │ ████░░░░░░ 72%
│ 17/20 hadir           │ Laporan berkualitas
│                                     │
│ Lembur Hari Ini        │ Target Aktivitas
│ ██░░░░░░░░ 20%       │ ████████░░ 82%
│ 4 orang lembur        │ 82/100 area
│                                     │
└─────────────────────────────────────┘
```

### Metrik Utama yang Ditampilkan

| Metrik | Arti | Target |
|--------|------|--------|
| **Kehadiran** | % pekerja yang hadir pada hari itu | 100% |
| **Ketepatan Waktu** | % kehadiran tepat waktu (tidak terlambat) | >90% |
| **Kualitas Laporan** | % laporan yang disetujui (tidak ditolak) | >80% |
| **Target Aktivitas** | % area yang dikerjakan vs target | >85% |
| **Produktivitas** | Rata-rata laporan per pekerja per hari | Minimal 3 laporan |

## Analitik Trend

### Grafik & Visualization

Dashboard menampilkan grafik untuk analisis trend:

**1. Kehadiran Trend (Bulanan)**
- Garis trend kehadiran per minggu
- Identifikasi pattern (ex: selalu rendah hari Jumat)

**2. Kualitas Laporan (Trend)**
- Persentase laporan disetujui vs ditolak
- Trend kualitas naik/turun

**3. Productivity (Aktivitas per Pekerja)**
- Rata-rata laporan per pekerja per minggu
- Perbandingan productivity antar pekerja

**4. Area Coverage (Heat Map)**
- Peta dengan warna menunjukkan intensitas coverage per area
- Identifikasi area yang sering/jarang dikerjakan

## Export Laporan

### Cara Export Laporan

1. Di halaman laporan, cari tombol **"Export"** atau **"Download"**
2. Pilih format:
   - **PDF** — Untuk cetak dan formal report
   - **Excel** — Untuk analisis lebih lanjut
3. Pilih periode (minggu/bulan/range tanggal)
4. Tap **"Download"** atau **"Generate"**
5. File akan tersimpan di perangkat Anda

### Menggunakan Export untuk Laporan Atasan

Data export bisa digunakan untuk:

- **Laporan ke Top Management** — Sampaikan ringkasan kinerja rayon
- **Evaluasi Pekerja** — Sebagai base untuk penilaian tahunan
- **Planning** — Analisis trend untuk planning improvement
- **Komunikasi** — Share dengan tim untuk transparency

## Filter & Customization

### Filter Laporan

Anda bisa filter laporan berdasarkan:

- **Periode** — Hari, minggu, bulan, custom range
- **Pekerja** — Laporan individual per pekerja
- **Area** — Kinerja per area
- **Tipe Aktivitas** — Per kategori pekerjaan (pembersihan, pemeliharaan, dll)

### Custom Report

Beberapa dashboard allow Anda membuat custom report:

1. **Pilih Metrik** — Kematian apa yang mau ditampilkan
2. **Pilih Periode** — Tanggal range
3. **Pilih Dimension** — Per area, per pekerja, per tipe
4. **Generate** — Sistem generate custom report

## Benchmarking & Target

### KPI (Key Performance Indicator)

Dashboard menampilkan KPI rayon Anda vs target/standar:

```
KPI RAYON PUSAT VS TARGET

                      Realisasi    Target    Status
Kehadiran             85%          95%       ❌ Below
Ketepatan Waktu       78%          90%       ❌ Below
Kualitas Laporan      82%          85%       ✓ Close
Target Aktivitas      82%          90%       ❌ Below
Lembur/Bulan          32 jam       50 jam    ✓ Good
```

### Analisis Deviation

Jika realisasi jauh dari target:

1. Cek laporan detail untuk identifikasi masalah
2. Lihat trend — apakah konsisten atau sesekali
3. Analisis sebab — ada issue tertentu, atau workload, atau skill
4. Planning improvement — action plan untuk naik ke target

## Komunikasi Hasil Laporan

### Feedback ke Tim

Setelah laporan selesai:

1. **Share dengan Tim** — Transparansi adalah kunci
2. **Highlight Achievement** — Apresiasi capaian positif
3. **Identify Issues** — Diskusikan area improvement
4. **Planning Improvement** — Buat action plan bersama
5. **Follow-up** — Cek apakah improvement strategy berhasil

### Laporan ke Manajemen

Laporan juga perlu dilaporkan ke Top Management:

1. **Simpilifikasi** — Buat summary untuk C-level (1-2 halaman)
2. **Focus on Business** — Bukan detail teknis, tapi impact
3. **Identifikasi Issues** — Apa yang perlu attention dari manajemen
4. **Rekomendasikan** — Apa yang butuh support/resource dari atasan

## Troubleshooting Dashboard

### "Data Laporan Tidak Update"

**Solusi:**
1. Refresh halaman (F5 atau pull-to-refresh)
2. Log out dan login lagi
3. Cek koneksi internet
4. Hubungi admin jika masih tidak update

### "Tidak Bisa Export Laporan"

**Solusi:**
1. Pastikan Anda punya role yang bisa export (Kepala Rayon, Top Management)
2. Cek koneksi internet stabil
3. Pilih period yang tidak terlalu besar (ex: 1 bulan instead of 1 tahun)
4. Hubungi admin jika masih gagal

### "Metrik Tidak Masuk Akal"

**Solusi:**
1. Cek apakah ada laporan yang belum divalidasi (status pending)
2. Lihat detail laporan untuk verifikasi
3. Hubungi Korlap jika ada data yang merasa salah

## Tips Menggunakan Dashboard

:::tip
**1. Cek Rutin**
- Lihat dashboard minimal 2x seminggu
- Pantau KPI dan trend
- Identifikasi issue early

**2. Deep Dive ke Detail**
- Jika ada KPI yang drop, cek laporan detail
- Lihat per pekerja untuk identifikasi problem
- Jangan hanya lihat angka aggregate

**3. Share Transparansi**
- Share laporan dengan tim
- Diskusikan trend dan challenge
- Planning improvement bersama

**4. Action Item**
- Jangan hanya lapor, tapi action
- Jika ada issue, bikin improvement plan
- Follow-up progress improvement

**5. Benchmark**
- Bandingkan dengan rayon lain (jika bisa)
- Lihat best practices dari rayon high performer
- Apply improvement ideas di rayon Anda
:::
