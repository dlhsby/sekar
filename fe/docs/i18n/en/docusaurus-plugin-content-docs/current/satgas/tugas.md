# View and Complete Tasks from Korlap

Complete guide to viewing, accepting, and finishing tasks given by Korlap (Field Coordinator).

## What is a Task?

**Task** is specific work assigned by Korlap to you. Each task has:

- **Description** — Detail of work to do
- **Location** — Area where task must be done
- **Priority** — Urgency level (Normal, Important, Urgent)
- **Deadline** — Time limit for completion
- **Status** — Pending, Accepted, In Progress, Done, etc

---

## Open Task List

### Method 1: From Menu

1. Tap **Menu** tab (☰)
2. Tap **"Tasks"**

If there's new task:

## Accept Task

### Steps to Accept Task

1. Open **task details** to accept
2. Tap **"ACCEPT TASK"** or **"ACCEPT"** button
3. System asks confirmation:
   ```
   Are you sure you accept this task?
   ```
4. Tap **"Yes, I Accept"**
5. Status changes to **"Accepted"**
6. You'll see notification:
   ```
   ✓ Task Accepted
   You can now start working on this task
   ```

### What Does Accepting Mean?

When you accept task:
- ✓ You commit to completing it
- ✓ Korlap knows you'll work on it
- ✓ Deadline starts counting
- ✓ You can start work

:::warning
**Don't accept task if unsure you can finish!** Better to reject and explain to Korlap.
:::

## Reject Task

### When to Reject Task?

Reject task if:
- ✗ No time available
- ✗ Don't have skill/equipment
- ✗ Other urgent work
- ✗ Not feeling well

### How to Reject Task

1. Open **task details** to reject
2. Tap **"REJECT"** button
3. System asks for reason:
   ```
   Reason for rejecting task?
   [Enter reason...]
   ```
4. Type your reason clearly
5. Tap **"Send"**
6. Task status becomes **"Rejected"**
7. Korlap sees rejection reason

:::note
Rejecting task is fine if you have clear reason. Communication important so Korlap can assign elsewhere.
:::

## Do Task Work

### After Accepting Task

After task status is **"Accepted"**, you can:

1. **Start working** — Do work per description
2. **Update status** — Change to "In Progress" when starting
3. **Document** — Take photos/video while working

### Update Task Status

When working, update status:

**Status 1: Start Work**
- Tap **"START WORK"** or **"START"** button
- Status becomes **"In Progress"**

**Status 2: Work Complete**
- After finish, tap **"MARK COMPLETE"** or **"COMPLETE"**
- Or make **activity report** (auto-links to task)

### Create Report for Task

**How to Make Report Linked to Task:**

1. Tap **Menu** → **Activities**
2. Tap **"+ Report Activity"**
3. In form, there's **"Related Task"** field
4. Select the task you're working on
5. Fill description and take photo
6. Tap **"SUBMIT"**

Report auto-links to task, and task marks **"Done"** after Korlap validates report.

## Task Status - Full Lifecycle

### Task Status Journey

```
[Not Accepted] → [Accepted] → [In Progress] → [Done]
        ↓
    [Rejected]
```

| Status | Meaning | Action |
|--------|---------|--------|
| **📋 Not Accepted** | New task from Korlap, waiting your answer | Accept or Reject |
| **✅ Accepted** | You accepted, not started | Start Work or Reject |
| **🔨 In Progress** | You're working | Mark Complete when done |
| **✓ Done** | Work finished, waiting Korlap approval | Wait for approval |
| **❌ Rejected** | You or Korlap rejected | Make new task or contact Korlap |

## Deadline & Notifications

### Task Deadline

Each task has **deadline (time limit)**:

```
Task: Clean Front Gate Area
Deadline: Today, 14:00
Time Left: 2 hours 30 minutes
```

### Deadline Notifications

You get notifications if:

- **Deadline Approaching** — 1 hour before deadline
  ```
  ⏰ Task "Clean Front Gate" deadline in 1 hour
  ```

- **Deadline Missed** — If past deadline
  ```
  ⚠️ Task "Clean Front Gate" past deadline
  ```

:::warning
If task past deadline, Korlap will know. Finish on schedule for best evaluation.
:::

### Extend Deadline

If need more time:

1. Contact **Korlap** directly (chat, call, or in-person)
2. Explain why you need more time
3. Korlap can extend deadline

**Don't just delay without talking!**

## Filter & Sort Tasks

### Filter Tasks

On task list page, you can filter:

**Filter 1: Status**
- Not Accepted
- Accepted
- In Progress
- Done
- Rejected

**Filter 2: Priority**
- Urgent (Red)
- Important (Yellow)
- Normal (Green)

**Filter 3: Date**
- Today
- This Week
- This Month
- All Time

### Sort Tasks

Sort by:

- **Nearest Deadline** — Closest deadline first
- **Newly Created** — Newest first
- **Highest Priority** — Urgent first

:::tip
**Best Practice**: Sort by "Nearest Deadline" so you don't miss deadlines.
:::

## Task Troubleshooting

### "Cannot Accept Task (Button Disabled)"

**Causes:**
- Status already changed (already accepted by someone)
- Task no longer active

**Solution:**
- Refresh page (pull to refresh)
- Contact Korlap if already accepted by someone

### "Task Not Showing in List"

**Possible:**
- Task filtered out (check active filters)
- Task for someone else, not you

**Solution:**
- Reset filters: tap **"Reset Filters"**
- Check area assignment with Korlap

### "Deadline Passed, Can I Still Submit?"

**Answer:**
- Can still submit, but recorded as "Late"
- Supervisor will see you finished outside deadline
- Explain or add report for context

### "Korlap Wants Revision, Do I Make New Report?"

**Yes**, follow steps:

1. See feedback/notes from Korlap in task details
2. Make **new activity report** with improvements per feedback
3. Link to same task in "Related Task" field
4. Submit revised report

## Task Work Tips

:::tip
**1. Accept or Reject Promptly**
- Don't delay, accept or reject right away
- Korlap needs to know who's working on it

**2. Communicate**
- If problem, contact Korlap directly
- Don't just submit if issue exists

**3. Document Well**
- Make clear activity report with photos
- Link to task for easy tracking

**4. On Time**
- Try to finish before deadline
- If can't, tell Korlap early

**5. Quality**
- Do work per task description
- Take good photos for report
:::
