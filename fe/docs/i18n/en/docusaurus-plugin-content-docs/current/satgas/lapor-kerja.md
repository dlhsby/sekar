# How to Create Work Reports & Activities

<p align="center"><img src="/img/illustrations/onb-photo.svg" alt="Photo verification illustration" width="240" /></p>

Complete guide to creating and submitting work reports with photos and videos.

## What is a Work Report?

**Work Report (Activity)** is documentation of your work containing:

- **Activity Description** — What you did
- **Activity Type** — Category of work (cleaning, maintenance, etc)
- **Photo/Video** — Visual proof of work
- **Work Location** — Area where work was done
- **Time** — When work was done

This report will be viewed and validated by supervisor/korlap for performance evaluation.

---

## Steps to Create Report

### 1. Open Activities Page

**Method 1 - From Menu:**
- Tap **Menu** tab (☰)
- Tap **"Activities"**

**Method 2 - From Home:**
- Tap **Home** tab
- Scroll down
- Tap **"+ Report Activity"** or **"Create Report"**

### 2. Activity Report Form

You will see a form with several fields:

```
┌──────────────────────────────────┐
│   ACTIVITY REPORT FORM          │
├──────────────────────────────────┤
│ Activity Type *                 │
│ [Select: Cleaning ▼]            │
│                                 │
│ Description *                  │
│ [Describe the work done...]     │
│                                 │
│ Location *                      │
│ [Location: Taman Bungkul ✓]     │
│                                 │
│ Activity Photo *                │
│ [+ TAKE PHOTO]                  │
│ [+ TAKE VIDEO]                  │
│                                 │
│ [SAVE DRAFT]  [SUBMIT]          │
└──────────────────────────────────┘
```

### 3. Select Activity Type

1. Tap the **"Activity Type"** dropdown
2. Select a category:
   - Cleaning
   - Maintenance
   - Repair
   - Planting
   - Other

3. Activity type will appear in field

:::note
Activity type must be selected to categorize your report.
:::

### 4. Fill in Work Description

1. Tap the **"Description"** field
2. Type a description of work you did:

**Example of Good Description:**
- "Cleaned Taman Bungkul area from plastic trash and dry leaves. Swept around playground and recreation area."
- "Trimmed tree branches sweeping over sidewalk in front of Taman Bungkul fence. Collected trimmings and took to waste collection point."

**Don't:**
- Too short ("Just worked")
- Too generic ("Did the task")
- No detail ("Whatever")

3. Minimum 20 characters for description

### 5. Check Location

Location will auto-fill from **your assigned work area**:

- Area: Taman Bungkul
- Coordinates: -7.295479, 112.762227

:::warning
**Location is fixed, cannot be changed** — matches your assigned area when you clocked in. If working in different area, contact supervisor.
:::

### 6. Take Photo Evidence

<p align="center"><img src="/img/mobile/lapor-kerja.png" alt="Work report screen" width="280" /></p>

*The activity report creation screen.*

System requires **minimum 1 photo or video** for work proof:

**How to Take Photo:**

1. Tap **"+ TAKE PHOTO"**
2. Allow camera access
3. Take photo of your work:
   - Photo of area before work
   - Photo of area after work
   - Photo of work results clearly

4. After photo, you can:
   - ✓ **Use** — Save this photo
   - ✗ **Retry** — Take another photo
   - 🔄 **Edit** — Crop or resize

5. Photo appears in preview

**How to Take Video:**

1. Tap **"+ TAKE VIDEO"** (if available)
2. Allow camera access
3. Record video of work (max 2-3 minutes)
4. After done, tap stop
5. Preview video and tap ✓ to save

:::tip
**Best Photo & Video Tips:**
- **Before & After**: Take photos of area before and after work
- **Good Lighting**: Daytime better than night
- **Clear & Bright**: Avoid blur or darkness
- **Show Context**: Display work area and results
- **Video**: Make short video (1-2 min) showing activity
:::

### 7. Add More Photos/Videos (Optional)

You can add up to **5-10 photos/videos**:

1. After first photo is saved, tap **"+ TAKE PHOTO AGAIN"**
2. Repeat for next photo
3. Photos display as gallery

## Save & Submit Report

### Option 1: Save as Draft

1. Fill all required fields
2. Tap **"SAVE DRAFT"**
3. Report will save on your device
4. You can edit and submit later

**When to Use Draft?**
- Work not finished yet
- Want to add photos/videos later
- Not sure about report now

### Option 2: Submit Report

1. Make sure all fields filled:
   - ✓ Activity Type
   - ✓ Description
   - ✓ Minimum 1 Photo/Video
   
2. Tap **"SUBMIT"** or **"SUBMIT"**

3. System will:
   - Upload photos/videos to server
   - Record time and location
   - Send notification to supervisor

4. You'll see message:
   ```
   ✓ Report Submitted
     ID: LAP-20260624-001
     Status: Awaiting Validation
   ```

5. Report sent to supervisor for validation

:::warning
**After Submitting**, report cannot be edited. Make sure everything correct before submit.
:::

## Report Status

After submitting, status changes based on validation process:

| Status | Meaning | Action |
|--------|---------|--------|
| **📝 Draft** | Not submitted, can still edit | Edit or Submit |
| **⏳ Awaiting Validation** | Submitted, supervisor reviewing | Wait for supervisor check |
| **✅ Approved** | Supervisor approved report | Done |
| **❌ Rejected** | Supervisor rejected with reason | Make new report |
| **📝 Revision** | Needs correction per notes | Edit per feedback |

### View Report Status

1. Open **Menu** → **Activities**
2. See list of your reports
3. Status shown for each item

## Editing Draft Report

### How to Edit Draft

If report still in **Draft** status:

1. Open **Menu** → **Activities**
2. Find report with "Draft" status
3. Tap report to open
4. Tap **"Edit"** or edit icon
5. Change fields you want
6. Tap **"Save"** or **"Submit"**

### Already Submitted Reports

If report already **submitted** (cannot edit directly):
- Wait for feedback from supervisor
- If rejected, you can make new report
- If revision needed, check supervisor notes and make new report with improvements

## Offline Mode

### Creating Report While Offline

You can still make reports while offline:

1. Take photo (saves locally)
2. Fill description
3. Tap **"SAVE DRAFT"**
4. Report saves on device

### Submitting Report While Offline

Reports cannot submit directly while offline. After connection returns:

1. Open draft report
2. Tap **"SUBMIT"**
3. System tries to upload
4. Status changes to "Awaiting Validation"

:::warning
Make sure connection stable when submitting report so photos upload properly.
:::

## Report Troubleshooting

### "Photo Too Large / Cannot Upload"

**Solution:**
- App auto-compresses photos
- If still fails, try taking again with lower resolution
- Reduce number of photos (make 2 reports if needed)

### "Report Rejected by Supervisor"

**What to Do:**

1. Check rejection reason in supervisor notes
2. Make new report with improvements:
   - More detailed description
   - Clearer photos
   - Follow supervisor feedback
3. Submit new report

### "Status Still 'Awaiting Validation' After 1 Day"

**Possible:**
- Supervisor hasn't checked report yet
- Technical issue

**Solution:**
- Contact supervisor to check status
- If needed, make additional report for today

### "Cannot Add Photo (Button Disabled)"

**Causes:**
- Reached maximum photo limit (usually 10)
- Report already submitted (cannot edit)

**Solution:**
- If photo limit, make new report for additional photos
- If submitted, wait for supervisor feedback

## Tips for Good Reports

:::tip
**1. Detailed Description**
- Say what you did, how, and results
- Example: "Cleaned recreation area from trash and dry leaves. Swept all areas, washed playground equipment with clean water."

**2. Quality Photos**
- Take minimum 2 photos: before and after
- Good light, clear object, not blurry
- Include surrounding area so supervisor knows location

**3. Match Activity Type**
- Pick type matching your work
- Don't be careless

**4. Timely**
- Make report same day work was done
- Don't pile up reports multiple times

**5. Honest & Accurate**
- Report what you actually did
- Don't fake photos or description
:::
