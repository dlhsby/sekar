# View Your Shift Schedule

Guide to viewing your work schedule (shifts) in the SEKAR app.

## What is a Shift Schedule?

**Shift Schedule** is your work plan that includes:

- **Work Hours** — What time to what time you work
- **Work Area** — Where you are assigned
- **Work Days** — Which days you work
- **Shift Type** — Shift 1 (morning), Shift 2 (afternoon), Shift 3 (night), etc

This schedule is created by admin/rayon head and you cannot change it yourself.

1. Tap **Home** tab
2. Scroll down to **"Shift Schedule"** section
3. You'll see this week's schedule

## Types of Shifts

### SEKAR Standard Shifts

SEKAR usually uses 3 shifts:

| Shift | Name | Hours | Description |
|-------|------|-------|-------------|
| **Shift 1** | Morning | 06:00 - 14:00 | Morning - afternoon |
| **Shift 2** | Afternoon/Evening | 14:00 - 22:00 | Afternoon - night |
| **Shift 3** | Night | 22:00 - 06:00 | Night - morning |

:::note
Shift hours may differ by rayon policy. Check your schedule for accurate times.
:::

### Days Off & Holidays

If no shift on a day, it shows **"OFF"**:

```
OFF
(No shift today)
```

Days off usually:
- Sundays
- Public holidays
- Scheduled off-days

## View Long-Term Schedule

### Weekly vs Monthly Schedule

**Default: Weekly Schedule**
- See 7 days (Monday - Sunday)
- Navigation buttons for previous/next week

**How to See Other Weeks:**

1. On Schedule page, find **"< Previous Week"** and **"Next Week >"** buttons
2. Tap button to navigate
3. Schedule changes to selected week

### Check Schedule Weeks Ahead

1. Open **Menu** → **Schedule**
2. Navigate to week you want
3. Or find **"View Monthly Schedule"** for full month overview

## Prepare Before Shift

### Pre-Shift Checklist

Based on your schedule, prepare:

1. **Night Before**
   - ✓ Check tomorrow's shift schedule
   - ✓ Make sure device fully charged
   - ✓ Prepare work equipment (tools, uniform, etc)

2. **Shift Morning**
   - ✓ Wake up on time (15-30 min before shift)
   - ✓ Be at location before shift starts
   - ✓ Make sure GPS active
   - ✓ Check internet connection

3. **When Clocking In**
   - ✓ Open SEKAR app
   - ✓ Tap **"CLOCK IN"**
   - ✓ Take selfie photo
   - ✓ Confirm

:::tip
**Arrive Early!** Come 5-10 minutes before shift to avoid being late and ensure GPS ready.
:::

## Schedule Changes

### If Schedule Changes

If admin/rayon head changes your schedule:

1. **Notification** — You'll get push notification
2. **Update App** — Open Schedule tab to see latest
3. **Refresh** — If not showing, pull-to-refresh or restart app

### Communicate Schedule Changes

If there's sudden schedule change:

1. **Contact Supervisor** — Reach out to Korlap/Rayon Head
2. **Confirm** — Make sure you understand new schedule
3. **Prepare** — Adjust preparation for new schedule

:::warning
Don't skip shift because you didn't see schedule update! Always check SEKAR before shift starts.
:::

## Schedule Management Tips

:::tip
**1. Set Alarm**
- Set alarm 30 minutes before shift
- Don't rely only on app notifications

**2. Write Down Schedule**
- Screenshot or write schedule in personal calendar
- If possible, write on home calendar

**3. Prepare Night Before**
- Prepare equipment at night
- Don't prepare in morning (might be late)

**4. Communicate with Team**
- Know who shifts with you
- Coordinate if changing areas

**5. Get Enough Rest**
- For Shift 3 (night), rest during day
- Don't work overtime multiple days without rest
:::

## Schedule Troubleshooting

### "Schedule Not Showing in App"

**Causes:**
- Schedule not entered by admin yet
- Data loading error
- Internet cut while loading

**Solution:**
1. Refresh page (pull-to-refresh)
2. Restart app
3. Make sure internet active
4. Contact admin if still not showing

### "App Schedule Different from What Supervisor Told Me"

**Solution:**
1. Contact supervisor to clarify
2. Usually app has most updated data
3. If mismatch, supervisor can update

### "Don't Know My Schedule"

**Solution:**
1. Open app → Schedule tab
2. See current week
3. Still confused? Contact supervisor
4. Can also ask teammate
