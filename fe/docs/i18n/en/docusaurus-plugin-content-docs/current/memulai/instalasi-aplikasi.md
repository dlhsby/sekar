# How to Install SEKAR Mobile App

Step-by-step guide to download and install the SEKAR app on your Android device.

## Device Requirements

### Minimum Specifications

- **OS**: Android 8.0 or newer
- **RAM**: 2 GB minimum (4 GB recommended)
- **Storage**: 100 MB free space
- **Connection**: WiFi or 4G/5G mobile data

### Required Permissions

The SEKAR app requires several permissions to work properly:

- **Location (GPS)** — For real-time tracking and GPS verification
- **Camera** — For photo verification during clock-in and work reports
- **Notifications** — To receive updates and reminders
- **Storage** — To save data and photos

## Installation Steps

### 1. Download APK File

:::note
Download from the official SEKAR page to ensure you get the latest version.
:::

**How to Download:**

1. Open a browser on your Android device (Chrome, Firefox, or other)
2. Visit: **https://sekar.wahyutrip.com/android**
3. Click the **"Download APK"** or **"Download"** button
4. File will start downloading (usually 50-100 MB)
5. Wait until download completes

:::tip
If download is slow, make sure your internet connection is stable or use WiFi.
:::

### 2. Installation Preparation

**Allow Apps from Unknown Sources:**

Since the APK file is not from Google Play Store, you need to grant permission:

1. Open **Settings** > **Security** or **Privacy**
2. Find **"Allow Unknown Apps"** or **"Install Unknown Apps"**
3. Select the browser app or file manager you used to download
4. Enable **"Allow from this source"** or **"Allow from this source"**

### 3. Install App

1. Open **File Manager** or **Downloads** on your device
2. Find the file named `sekar-*.apk` (example: `sekar-1.0.0.apk`)
3. Tap the file to start installation
4. Click **"Install"** or **"Install"**
5. Wait until installation completes (usually 30-60 seconds)
6. Click **"Open"** or **"Open"** to launch the app

### 4. Initial Configuration

**First Time Login:**

1. The SEKAR app will open to the login page
2. Enter your phone number and password
3. Click **"LOGIN"**

**Grant Permissions:**

When you first login, the app will request several permissions:

- **Location Permission (GPS)**: Tap **"Allow"** to enable GPS
- **Camera Permission**: Tap **"Allow"** to use camera
- **Notification Permission**: Tap **"Allow"** to receive notifications

:::warning
**These permissions are important for app functionality.** If you don't grant them now, you won't be able to use features like clock-in, work reports, or receive notifications.
:::

### 5. Verify Installation

After successful login:

1. You'll see the **"Home"** or **"Home"** page
2. Check connection status at the top (should show **"ONLINE"** with green indicator)
3. The app is ready to use!

## Enabling GPS

### GPS is Critical for SEKAR

The SEKAR app **requires GPS to be active** for:

- Location verification during clock-in
- Real-time worker tracking
- Accurate work reports with location

### How to Enable GPS

1. Open **Settings** > **Location**
2. Enable **"Location"** or **"Location Services"**
3. Select mode: **"High Accuracy"** or **"High Accuracy"** (GPS + WiFi combination)
4. Open SEKAR app again

:::note
**Want to Save Battery?** "Battery Saver" mode uses WiFi and mobile data only (no GPS), but GPS accuracy decreases. Use **"High Accuracy"** for best results.
:::

## Updating the App

### How to Update to Latest Version

1. Visit: **https://sekar.wahyutrip.com/android**
2. Check your app version in **"Profile"** > **"Diagnostics"**
3. If a newer version is available on the page, download and install the new APK file
4. Installation will automatically replace the old version

:::tip
The app will remind you if a newer version is available. Always update for latest features and security.
:::

## Uninstalling the App

If you want to remove SEKAR:

1. Open **Settings** > **Apps**
2. Find **"SEKAR"** in the app list
3. Tap the app, then click **"Uninstall"**
4. Confirm removal
5. App data will be deleted from your device

:::warning
**Important Note**: Uninstalling the app will delete all local data. If you have reports or offline data not synchronized, make sure to upload them to the server before uninstalling.
:::

## Installation Troubleshooting

### "APK File is Corrupted or Invalid"

**Solution:**
- Delete the downloaded APK file
- Download again from the official page
- Make sure download completes 100% before installing
- Try downloading using WiFi instead of mobile data

### "Not Enough Storage Space"

**Solution:**
- Open **Settings** > **Storage**
- Delete unnecessary apps or files
- Clear cache of other apps
- Try installing again

### "Permission to Install from Unknown Source is Blocked"

**Solution:**
- Open **Settings** > **Security**
- Enable **"Allow Unknown Apps"**
- Select the browser app you used to download
- Try installing again

### "App Crashes or Won't Open"

**Solution:**
- Restart your device
- Delete and reinstall SEKAR
- Make sure your Android is up-to-date
- Contact admin if problem continues
