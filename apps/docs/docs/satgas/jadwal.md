# Melihat Jadwal Shift Anda

Panduan untuk melihat jadwal kerja (shift) Anda di aplikasi SEKAR.

## Apa itu Jadwal Shift?

**Jadwal Shift** adalah rencana kerja Anda yang mencakup:

- **Jam Kerja** — Jam berapa sampai jam berapa Anda bekerja
- **Area Kerja** — Di mana Anda ditugaskan
- **Hari Kerja** — Hari apa saja Anda bekerja
- **Tipe Shift** — Shift 1 (pagi), Shift 2 (siang), Shift 3 (malam), dll

Jadwal ini dibuat oleh admin/kepala rayon dan tidak bisa Anda ubah sendiri.

---

## Melihat Jadwal Anda

### Cara 1: Dari Menu

<p align="center"><img src="/img/mobile/jadwal.png" alt="Jadwal shift" width="280" /></p>

*Tampilan jadwal shift mingguan.*

1. Tap tab **Menu** (☰)
2. Tap **"Jadwal"**
3. Jadwal Anda akan ditampilkan

### Cara 2: Dari Beranda

1. Tap tab **Beranda**
2. Scroll ke bawah hingga section **"Jadwal Shift"**
3. Anda akan melihat jadwal minggu ini

---

## Format Jadwal

### Tampilan Jadwal

Jadwal ditampilkan dalam format **weekly calendar** (kalender mingguan):

```
┌──────────────────────────────────────┐
│         JADWAL SHIFT MINGGU INI      │
├──────────────────────────────────────┤
│                                      │
│ SEN  SEL  RAB  KAM  JUM  SAB  MIN   │
│
│ SHIFT 1          SHIFT 1          
│ 06:00 - 14:00    06:00 - 14:00    
│ Taman Bungkul    Taman Bungkul    
│
│ SHIFT 2          SHIFT 2         LIBUR
│ 14:00 - 22:00    14:00 - 22:00   
│ Taman Bungkul    Taman Bungkul   
│
│ SHIFT 3          SHIFT 1         SHIFT 3
│ 22:00 - 06:00    06:00 - 14:00   22:00 - 06:00
│ Area Parkir      Taman Bungkul   Area Parkir
│
└──────────────────────────────────────┘
```

### Informasi Per Shift

Setiap kotak shift menampilkan:

- **Nama Shift** — Shift 1, Shift 2, Shift 3, dll
- **Jam Kerja** — Jam mulai - jam selesai
- **Area** — Lokasi area kerja
- **Status** — Libur (jika tidak ada shift)

- Radius toleransi GPS (biasanya ±100m)
- Jam istirahat (jika ada)
- Catatan khusus dari supervisor

## Jenis-Jenis Shift

### Shift Standar SEKAR

SEKAR biasanya menggunakan 3 shift:

| Shift | Nama | Jam | Deskripsi |
|-------|------|-----|-----------|
| **Shift 1** | Pagi | 06:00 - 14:00 | Pagi - siang |
| **Shift 2** | Siang/Sore | 14:00 - 22:00 | Siang - malam |
| **Shift 3** | Malam | 22:00 - 06:00 | Malam - pagi |

:::note
Jam shift bisa berbeda tergantung kebijakan masing-masing rayon. Lihat jadwal Anda untuk jam yang akurat.
:::

### Libur & Off-Days

Jika ada hari tanpa shift, akan ditampilkan **"LIBUR"**:

```
LIBUR
(Tidak ada shift hari ini)
```

Libur biasanya:
- Hari Minggu
- Hari raya/libur nasional
- Off-day yang dijadwalkan

## Melihat Jadwal Jangka Panjang

### Jadwal Mingguan vs Bulanan

**Default: Jadwal Mingguan**
- Lihat 7 hari (Senin - Minggu)
- Tombol navigasi untuk minggu sebelum/sesudah

**Cara Lihat Minggu Lain:**

1. Di halaman Jadwal, cari tombol **"< Minggu Sebelumnya"** dan **"Minggu Berikutnya >"**
2. Tap tombol untuk navigasi
3. Jadwal akan berganti ke minggu yang dipilih

### Cek Jadwal Beberapa Minggu ke Depan

1. Buka tab **Menu** → **Jadwal**
2. Navigasi ke minggu yang ingin dilihat
3. Atau cari fitur **"Lihat Jadwal Bulanan"** untuk overview bulan penuh

## Persiapan Sebelum Shift

### Checklist Sebelum Masuk Shift

Berdasarkan jadwal Anda, siapkan:

1. **Hari Sebelumnya**
   - ✓ Cek jadwal shift besok
   - ✓ Pastikan perangkat fully charged
   - ✓ Siapkan perlengkapan kerja (tools, uniform, dll)

2. **Pagi Shift Mulai**
   - ✓ Bangun sesuai waktu (15-30 menit sebelum jam shift)
   - ✓ Siap di lokasi sebelum jam shift dimulai
   - ✓ Pastikan GPS aktif
   - ✓ Periksa koneksi internet

3. **Saat Absen Masuk**
   - ✓ Buka aplikasi SEKAR
   - ✓ Tap **"MASUK"**
   - ✓ Ambil foto selfie
   - ✓ Konfirmasi

:::tip
**Datang Lebih Awal!** Datang 5-10 menit sebelum shift untuk menghindari terlambat dan memastikan GPS siap.
:::

## Perubahan Jadwal

### Jika Jadwal Berubah

Jika admin/kepala rayon mengubah jadwal Anda:

1. **Notifikasi** — Anda akan dapat notifikasi push
2. **Update Aplikasi** — Buka tab Jadwal untuk lihat jadwal terbaru
3. **Refresh** — Jika tidak muncul, pull-to-refresh atau restart aplikasi

### Komunikasi Perubahan Jadwal

Jika ada perubahan jadwal yang tiba-tiba:

1. **Hubungi Supervisor** — Segera hubungi Korlap/Kepala Rayon
2. **Konfirmasi** — Pastikan Anda mengerti jadwal baru
3. **Persiapan** — Sesuaikan persiapan untuk jadwal baru

:::warning
Jangan skip shift karena tidak lihat update jadwal! Selalu cek aplikasi SEKAR sebelum shift dimulai.
:::

## Tips Mengatur Shift

:::tip
**1. Set Alarm**
- Set alarm 30 menit sebelum shift
- Jangan bergantung hanya notifikasi aplikasi

**2. Catat Jadwal**
- Screenshot atau catat jadwal di kalender pribadi
- Jika bisa, tulis di papan tulis di rumah

**3. Persiapan Malam Sebelumnya**
- Siapkan perlengkapan malam hari
- Jangan persiapan pagi hari karena bisa terlambat

**4. Komunikasi dengan Tim**
- Tahu siapa yang shift bersama Anda
- Koordinasikan jika ada perpindahan area

**5. Istirahat Cukup**
- Untuk shift 3 (malam), istirahat siang
- Jangan lembur berhari-hari tanpa istirahat
:::

## Troubleshooting Jadwal

### "Jadwal Tidak Muncul di Aplikasi"

**Penyebab:**
- Jadwal belum dimasukkan admin
- Ada error loading data
- Koneksi internet putus saat loading

**Solusi:**
1. Refresh halaman (pull-to-refresh)
2. Restart aplikasi
3. Pastikan koneksi internet aktif
4. Hubungi admin jika masih tidak muncul

### "Jadwal di Aplikasi Berbeda dengan yang Diberitahu Supervisor"

**Solusi:**
1. Hubungi supervisor untuk klarifikasi
2. Biasanya aplikasi punya data paling updated
3. Jika ada ketidaksesuaian, supervisor bisa update

### "Tidak Tahu Jadwal Saya Apa"

**Solusi:**
1. Buka aplikasi → Tab Jadwal
2. Lihat minggu saat ini
3. Jika masih bingung, hubungi supervisor
4. Bisa juga tanya teman setim Anda
