# Request and Report Overtime (Overtime)

<p align="center"><img src="/img/illustrations/illo-shifts.svg" alt="Shifts illustration" width="240" /></p>

Complete guide to requesting overtime and reporting your overtime work.

## What is Overtime?

**Overtime** is extra work outside normal hours. Every overtime must:

- **Be approved first** by supervisor/Korlap
- **Be documented** with work report
- **Be paid/compensated** per policy

---

## Request Overtime

### When Can You Request Overtime?

You can request overtime if:
- ✓ Urgent work not finished during normal shift
- ✓ Special project needed outside hours
- ✓ Special event or activity at work area

:::warning
**Cannot just overtime anytime!** Overtime must have clear reason and supervisor approval.
:::

### Steps to Request Overtime

#### 1. Open Overtime Page

**Method 1 - From Menu:**
- Tap **Menu** tab (☰)
- Tap **"Overtime"**

**Method 2 - From Home:**
- Tap **Home** tab
- Tap **"REQUEST OVERTIME"** button (if available)

#### 2. Overtime Request Form

<p align="center"><img src="/img/mobile/lembur.png" alt="Overtime request form" width="280" /></p>

*The overtime request submission screen.*

You'll see a form:

```
┌──────────────────────────────────┐
│   REQUEST OVERTIME FORM         │
├──────────────────────────────────┤
│ Date *                          │
│ [Select Date: June 24, 2026 ▼]  │
│                                 │
│ Overtime Start Time *           │
│ [17:00 ▼] (After normal shift)  │
│                                 │
│ Overtime End Time *             │
│ [20:00 ▼]                       │
│                                 │
│ Duration: 3 hours               │
│                                 │
│ Reason for Overtime *           │
│ [Describe reason for overtime...]│
│                                 │
│ [REQUEST]  [CANCEL]            │
└──────────────────────────────────┘
```

#### 3. Fill Date & Overtime Time

**Select Date:**
- Tap **"Date"** field
- Pick overtime date (which day)

**Fill Start & End Times:**
- **Start Time** — When overtime starts (ex: 17:00)
- **End Time** — When overtime ends (ex: 20:00)
- System auto-calculates duration (3 hours in example)

:::note
Start time usually after normal shift ends (ex: 14:00 or 17:00 depending on shift).
:::

#### 4. Fill Overtime Reason

Type your reason clearly:

**Example Good Reasons:**
- "Complete playground area cleaning not finished afternoon shift. Area needs cleaning before weekend for visitor safety."
- "Help fix broken light in park area to maintain evening visitor comfort."
- "Special event setup for community activity tomorrow morning. Need preparation tonight."

#### 5. Send Overtime Request

1. Check all fields filled
2. Tap **"REQUEST"** button
3. System sends to supervisor
4. You'll see message:
   ```
   ✓ Overtime Request Sent
   Waiting for supervisor approval
   ```

### Overtime Request Status

After requesting:

| Status | Meaning | Action |
|--------|---------|--------|
| **⏳ Pending** | Waiting for supervisor approval | Wait for answer |
| **✅ Approved** | Supervisor approved overtime | Do overtime as planned |
| **❌ Rejected** | Supervisor not approved | Contact supervisor for reason |

## Do Overtime

### Before Starting Overtime

1. **Make Sure Approved** — Check status is "Approved"
2. **Prepare** — Get needed equipment/tools
3. **Communicate** — Know which supervisor on-duty

### During Overtime

Overtime done **outside normal shift hours**, but:

- ✓ Must **keep GPS recording** (location tracking active)
- ✓ Must **make work report** for documentation
- ✓ Don't need clock in/out (not in normal shift hours)

---

## Report Overtime

### Create Overtime Report

After overtime finished, **make work report**:

#### 1. Open Activity Report

1. Tap **Menu** → **Activities**
2. Tap **"+ REPORT ACTIVITY"** or **"Create New Report"**

#### 2. Fill Report Form

```
Activity Type: [Select appropriate]
Description: "Overtime - [Describe overtime work]"
Location: [Area where overtime was done]
Time: [Overtime time (ex: 17:00 - 20:00)]
Photo: [Take photo of overtime work results]
```

#### 3. Link to Overtime

In report form, there's **"Related Overtime"** field:

1. Tap **"Related Overtime"** field
2. Select overtime matching this report
3. Overtime links to report

#### 4. Take Photo & Submit

1. Take photo of overtime work results
2. Tap **"SUBMIT"**
3. Report sent to supervisor

### Example Overtime Report

```
Activity Type: Repair
Description: Overtime - Completed fixing light 
in playground area. Cleaned up work area and 
made sure area safe for visitors.
Location: Taman Bungkul
Time: 17:00 - 20:00
Photo: [Photo of area before & after repair]
Related Overtime: June 24, 2026 Overtime (17:00-20:00)
```

## Offline Mode Overtime

### Request Overtime While Offline

If offline when requesting:

1. Fill overtime form normally
2. Tap **"REQUEST"**
3. Form saves as draft locally
4. When connected, system auto-submits

### Report Overtime While Offline

You can make overtime report offline too:

1. Take photo (saves locally)
2. Fill activity report
3. Link to overtime
4. Tap **"SUBMIT"** when online

:::warning
Make sure overtime data **syncs before overtime ends**, so supervisor can validate on time.
:::

## Overtime Troubleshooting

### "Overtime Request Rejected, Why?"

**Possible Reasons:**
- Supervisor schedule doesn't allow approval
- Different overtime policy that day
- Supervisor not available then

**Solution:**
1. Check feedback/reason from supervisor in app
2. Contact supervisor directly to discuss
3. Request overtime at different time

### "Overtime Status Still 'Pending' After Long Time"

**Possible:**
- Supervisor hasn't checked request
- Technical issue

**Solution:**
- Chat/contact supervisor to check
- If urgent, communicate directly

### "Already Did Overtime But Forgot to Report"

**Must Report Soon:**

1. Make activity report (even if not 24 hours yet)
2. Describe overtime work done
3. Link to matching overtime
4. Take photo if possible (or report area condition)

### "Overtime Report Rejected"

**What to Do:**

1. Check supervisor feedback about rejection reason
2. Make new report with improvements (better photos, clearer description)
3. Link to same overtime
4. Resubmit

## Overtime Tips

:::tip
**1. Request Before Overtime**
- Don't overtime first then request
- Request first to get approval

**2. Clear Reason**
- Explain overtime reason with detail
- Supervisor needs to know why

**3. Good Documentation**
- Make detailed report
- Take photos showing overtime work

**4. Don't Overdo**
- Overtime shouldn't hurt your health
- Rest enough for next shift

**5. Communicate**
- Tell supervisor if overtime can't finish as planned
- Document what you already did
:::
