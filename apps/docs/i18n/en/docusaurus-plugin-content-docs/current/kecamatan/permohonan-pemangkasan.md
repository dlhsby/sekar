# Submitting Tree Pruning Request

<p align="center"><img src="/img/illustrations/illo-reports.svg" alt="Reports illustration" width="240" /></p>

Guide for District Staff to submit tree pruning requests through SEKAR.

## What is Tree Pruning Request?

**Tree Pruning Request (Pruning Request)** is a system for district staff (public/external) to request tree pruning that is:

- Hazardous (branches sweeping sidewalk/road)
- Problematic (broken branch, disease)
- Needs maintenance (overgrown foliage)

Requests will be reviewed and scheduled by Environmental Department for completion.

## Steps to Submit Request

### 1. Fill in Requestor Identity

**Full Name:**
- Requestor name or organization submitting
- Example: "District Wonokromo" or "Bapak Ahmad Wijaya"

**Contact:**
- Email — for correspondence
- Phone — for quick follow-up

### 2. Determine Tree Location

**District/Village:**
- Select from dropdown or type name

**Address Detail:**
- Write complete address
- Example: "Jl. Semeru No. 5, in front of SDN Semeru, Wonokromo"

**GPS Coordinates:**
1. Tap **"Choose Location on Map"**
2. Map will open
3. Pan and zoom to tree location
4. Tap location on map to pin (marker)
5. Confirm location

Or, if you know coordinates:
- Latitude: -7.295479
- Longitude: 112.762227

### 3. Describe Problem

Type detailed description of tree problem:

**Good Description Example:**
```
"Mango tree in front of District office has large 
branch sweeping sidewalk. This branch disturbs 
pedestrians and may be dangerous during strong wind. 
Request to prune so branch doesn't sweep sidewalk 
anymore."
```

**Include:**
- Tree type (if you know)
- Problem (hazardous, disease, overgrown)
- Current condition
- What needs to be done

### 4. Upload Evidence Photos

Take or upload tree photos:

1. Tap **"+ TAKE/UPLOAD PHOTO"**
2. Choose camera or gallery
3. Take/select tree photo
4. Upload minimum 1 photo (ideal 2-3 photos)

**Good Photos:**
- Show entire tree
- Show problematic part (branch sweeping, broken, etc)
- Good lighting, clearly visible

### 5. Set Priority

**Urgent (Hazardous):**
- Broken branch can fall
- Branch directly blocking road/sidewalk
- Safety risk

**Normal (Maintenance):**
- Routine maintenance
- Overgrown foliage
- Seasonal preparation

:::warning
If safety risk exists (broken branch, tilted tree), choose **URGENT** for quick handling.
:::

### 6. Send Request

1. Verify all information is correct
2. Tap **"SEND REQUEST"** or **"Submit"**
3. System will send request
4. You'll see confirmation:
   ```
   ✓ Request Sent!
   
   Request Number: REQ-2026-06-001
   Status: Received (Awaiting Verification)
   
   We will contact you to confirm location
   and pruning schedule within 1-2 business days.
   
   Reference Number: REQ-2026-06-001
   ```

## Communication & Follow-up

### You Will Be Contacted

After submitting request:

1. **Email/SMS** — Submission confirmation (within 24 hours)
2. **Phone** — Verify location and schedule (within 1-2 business days)
3. **Notification** — Status updates via email/SMS

### Information That May Be Asked

- Location clarification (may request GPS or more detail)
- Tree type (if not recorded)
- Flexible schedule (reschedule request)
- On-site contact when team arrives

## Pruning Request FAQ

**Q: How long does request take?**
A: From submission to completion usually 2-4 weeks, depending on complexity and priority queue.

**Q: Is it free?**
A: Yes, this is a public service from Environmental Department.

**Q: Can I request urgent?**
A: Yes, if safety risk (broken branch, fallen tree), choose urgent for priority handling.

**Q: If tree is on private property?**
A: Request accepted, but team will coordinate with property owner for access.

**Q: Can I request specific schedule?**
A: Try communicating preference when team contacts for location verification.

## Troubleshooting Request

### "Request Not Submitted"

**Solution:**
1. Make sure all required fields filled (marked *)
2. Wait 5-10 seconds after tap submit
3. Check internet connection
4. Refresh page and try again

### "No Response After 3 Days"

**Solution:**
1. Check status in app (may already be scheduled)
2. Check email/SMS for notifications
3. Contact Environmental Department hotline: [contact admin for number]
4. Refer to your request number

### "Want to Cancel Request"

**How:**
1. If status "Awaiting Verification", can cancel from app
2. If already "Scheduled", contact team to reschedule/cancel
3. Include request reference number

## Tips for Effective Request

:::tip
**1. Detailed Description**
- Explain problem specifically
- Include recognizable landmark location
- Attach clear photos

**2. Quality Photos**
- Take photos in daytime with good light
- Show entire tree and problem
- If possible, photos from different angles

**3. Accurate Information**
- Location must be accurate (use map if possible)
- Active phone number for follow-up
- Valid email for notifications

**4. Wait for Confirmation**
- Don't assume immediate work
- Wait for team to contact for verification
- Confirm schedule if asked

**5. Documentation**
- Save request reference number
- Note submission and follow-up dates
- Keep communication records
:::
